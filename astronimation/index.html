Cara Kelola Soal

Setiap game MPI memiliki panel admin bawaan untuk mengelola soal langsung dari browser — tanpa perlu coding!

💡 Fitur kelola soal tersedia di dalam game. Kamu bisa menambah, mengedit, dan menghapus soal kapan saja.
1️⃣
Buka game lalu klik tombol "Kelola Soal"
Biasanya ada di pojok kiri bawah layar game.
2️⃣
Masukkan password admin
Password default: guruhebat
3️⃣
Tambah, edit, atau hapus soal
Soal tersimpan otomatis di browser. Gunakan fitur ekspor JSON untuk memindahkan soal ke perangkat lain.
